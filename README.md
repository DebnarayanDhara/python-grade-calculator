#  Grade Calculator

A Python-based grade calculator that evaluates marks and assigns grades with pass/fail results.

##  Features
- User name and marks input system
- Automatic grade calculation (Ex, A, B, C, D, F)
- Pass/Fail evaluation
- Handles invalid inputs

## 📊 Grade Criteria

| Marks Range | Grade |
|------------|------|
| 90 - 100   | Ex   |
| 80 - 89    | A    |
| 70 - 79    | B    |
| 60 - 69    | C    |
| 50 - 59    | D    |
| Below 50   | F    |
